=====================================
Employee Tags: Restricted Edit Access
=====================================

* Disables quick create functionality for the tags field of Employee
* Adds a new group for restricting who can edit the tags

Installation
============
* Just install this module

Configuration
=============
* Add users to the new "Employee Tag Editor" group
* Configure the tags themselves at Employees -> Configuration -> Employee Tags
